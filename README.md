_IT University of Copenhagen, Technical Interaction Design (Autumn 2024), KSTEIND1KU_

DogPal is a responsive web platform designed to help dog owners connect, organize meetups, and build local communities around their dogs. The platform makes it easy to discover events, create playdates, and manage dog profiles — all in one dedicated space focused entirely on dog-related social interaction.

Developed as part of the Technical Interaction Design course at ITU, the project is based on user research and aims to support both social and practical needs of dog owners.

**✨ Key Features**
- Browse and join dog-related events
- Create and manage meetups and playdates
- User profiles with registered dogs
- Event comments for communication and coordination
  
**How to get up and running:**
1. `npm install`
2. `npm run dev`
